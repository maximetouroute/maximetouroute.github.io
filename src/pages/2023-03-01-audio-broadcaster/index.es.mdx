Audio Broadcaster es un software compartido con lugares culturales, músicos, artistas digitales y, en general, con la comunidad creativa. 
Permite transmitir audio en vivo a smartphones de manera muy ligera, sin necesidad de hardware específico y sin necesidad de instalar una aplicación. 
El software se actualiza y mejora __según sus necesidades__ para desbloquear una amplia variedad de casos de uso.

## Características 

- Audio estéreo de alta definición
- Latencia ultra baja {'<'}100 ms en 4G, {'<'}10 ms en Wi-Fi
- Funciona en 4G (tanto para el público como para el emisor)
- Capacidad: 100+ participantes
- No se necesita hardware específico (computadora, smartphones)
- Alcance: ilimitado (en todo el mundo, siempre que haya acceso a internet)

## ¿Cómo funciona?

Use un teléfono o una computadora para transmitir audio. El audio puede provenir de un micrófono o de un software (Media Player, Ableton Live, ...). El sonido se transmite a través de la red de internet (4G o Wi-Fi). El público abre una página web (o escanea un código QR) y recibe la señal de audio en vivo de inmediato. No es necesario instalar una aplicación.



## Casos de uso

* Visitas guiadas
